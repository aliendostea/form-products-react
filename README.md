# Dashboard CRUD for ecommerce products - React TS Vite ⚡️

## How to start this project? 📋

```bash
# Install dependencies
$ yarn install

# Start development server
$ yarn dev
´
# Start build
$ yarn build

# Start eslint
$ yarn lint
```
